Data and descriptive statistics

Mean

The mean is found by adding all the values and dividing with the
total amount of values

$$3,8,5,3,4,9,0,3,2,10=\frac {3+8+5+3+4+9+0+3+2+10} {10}=4.7$$

Median

The median is found by rearranging all the values in order
and finding the middle value

$$0,9,5,7,4$$

$$=0,4,5,7,9=5$$

But when there is no middle value you need to add the values next
to the middle point and divide by 2

$$0,2,3,3,3,*,4,5,8,9,10 = \frac {3+4} {2}=3.5$$

Mode

mode is the most common value

$$3,8,5,3,4,9,0,3,2,10 = 3$$

Range

Range is Subtracting the largest value with the smallest value

$$3,8,5,3,4,9,0,3,2,10 = 10-0=10$$

Upper and Lower Quartile

Finding the Quartiles can be done by first finding the middle Quartile
(also known as the median) and then finding the middle of the split values

$$0,2,3,3,3,*,4,5,8,9,10$$

lower Quartile: $$0,2,3,3,3 = 3$$

upper Quartile: $$4,5,8,9,10 = 8$$

Standard Deviation

To calculate the sample Standard Deviation this formula can be done

$$\sqrt{ \frac {\sum(x-\bar{x})^{2}} {n-1} }$$

Firstly, the mean needs to be found

$$\bar{x}=\frac{sum}{n}=\frac{81+93+98+89+88}{5}=\frac{450}{5}=90$$

Then subtract each value and square them

$$\sqrt{\frac {(81-90)^{2}+(93-90)^{2}+(98-90)^{2}+(89-90)^{2}+(88-90)^{2}} {5-1}}$$

tip: if you add all the subtracted values the answer will be 0

$$(-8)+(3)+(8)+(-1)+(-2)=0$$

$$\sqrt{ \frac {(-8)^{2}+(3)^{2}+(8)^{2}+(-1)^{2}+(-2)^{2}} {4} }=5.958$$
